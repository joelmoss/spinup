Row Actions

  • ↻ Restart — restart the process (highlighted red on errored rows)
  • ■ Stop — stop a running process
  • ↗ Show — focus the terminal/agent in its VS Code window
  • Actions are contextual: running processes get stop/restart, idle get show, errored get restart/show
